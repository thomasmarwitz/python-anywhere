# Hello World Flask App

## Prerequisites
- Python 3.11 installed

## Setup (macOS/Linux)
1. Create and activate the virtual environment:

   ```bash
   python3.11 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:

   ```bash
   pip install flask
   ```

## Run
```bash
python app.py
```

Open http://127.0.0.1:5000 in your browser.
